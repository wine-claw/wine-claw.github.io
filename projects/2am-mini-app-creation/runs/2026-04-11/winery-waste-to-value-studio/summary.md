# Winery Waste-to-Value Studio

## What was built
A mobile-first static mini-app that helps a winery explore plausible circular-economy pathways for grape marc, lees, wastewater, and prunings.

## What works
- Stream selection for marc, lees, wastewater, and prunings
- Context controls for scale, CAPEX appetite, complexity tolerance, value goal, and infrastructure readiness
- Presets for small estate, mid-range producer, large winery, and sustainability-led use cases
- Ranked pathway cards with rationale, cautions, and "what would need to be true" notes
- Custom SVG flow visual that updates with the current selection
- Local hero image and experimental warning banner
- Static HTML/CSS/JS only, no framework or build step

## Known limitations
- Screening/education only, not a validated engineering or commercial decision tool
- Uses qualitative scoring rather than real economics, treatment design, logistics models, or region-specific rules
- Pathway library is curated, not exhaustive
- No manual phone test yet

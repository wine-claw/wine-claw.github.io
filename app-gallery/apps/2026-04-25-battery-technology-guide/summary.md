# Battery Technology Guide

Interactive single-file mini-app covering battery technology history and modern/future chemistries.

## Built

- `index.html` — self-contained HTML/CSS/JS app with no external dependencies.
- Dark-mode scientific/professional responsive design.
- Searchable, scrollable, clickable battery history timeline.
- Technology comparison selector for lead-acid, zinc-carbon, nickel-cadmium, nickel-metal hydride, lithium-ion, sodium-ion, solid-state, and lithium-sulfur batteries.
- SVG animated energy-density bar chart.
- Responsive comparison table with energy density, cycle life, cost, safety, status, and applications.
- Future batteries section covering solid-state, sodium-ion, and lithium-sulfur.
- References section with source citations and links.

## Verification

- HTML file created as a single self-contained document.
- Uses embedded CSS and JavaScript only.
- Uses inline SVG for all charts/visuals; no external images or libraries.
